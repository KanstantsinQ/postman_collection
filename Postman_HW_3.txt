===============

Pobrać listę walut.

Iterować po liście walut.

W każdej iteracji wysyłać zapytanie do serwera w celu pobrania kursu każdej waluty.

Jeśli zwrócony zostanie kod 500, przechodzimy do następnej iteracji.

Jeśli otrzymamy kod 200, sprawdzamy response JSON pod kątem obecności pola "Cur_OfficialRate".

Jeśli pole istnieje, wypisujemy do konsoli informację o walucie w formie odpowiedzi:

JSON
{
    "Cur_Abbreviation": str,
    "Cur_ID": int,
    "Cur_Name": str,
    "Cur_OfficialRate": float,
    "Cur_Scale": int,
    "Date": str
}
Przechodzimy do następnej iteracji.
